# CP-063 — Product Evolution Rules

**Estado:** Draft
**Versión:** 1.0
**Owner:** Product

---

# Objetivo

Definir reglas para la evolución del producto.

---

# Toda nueva funcionalidad debe responder

¿Hace que el usuario tome mejores decisiones gastronómicas?

---

# Toda nueva funcionalidad debe

- mejorar el Brain;
- mejorar la experiencia;
- reducir fricción;
- aumentar confianza.

---

# Toda nueva funcionalidad NO debe

- aumentar complejidad;
- duplicar otra;
- existir por moda;
- romper la simplicidad.

---

# Regla de Oro

Cuando haya dudas.

Elegir siempre la versión más simple.

---

# Estado

Este documento actúa como filtro para toda evolución futura del producto.
