# Objetivo

Definir cuándo vale la pena interrumpir al usuario.

Las notificaciones deben aportar valor.

Nunca generar ruido.

---

# Filosofía

Cada notificación debe responder.

¿Por qué vale la pena recibirla?

Si no existe una buena respuesta.

No se envía.
