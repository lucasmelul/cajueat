# CP-044 — Conversation Memory

**Estado:** Draft
**Versión:** 1.0
**Owner:** Product

---

# Objetivo

Definir cómo el Brain recuerda una conversación.

---

# Filosofía

Durante una conversación.

El usuario nunca debería repetir información.

---

# Ejemplo

Usuario.

Quiero sushi.

↓

Después.

Algo tranquilo.

↓

El Brain entiende que sigue hablando de sushi.

---

# Memoria de Conversación

Debe recordar.

- intención
- contexto
- ubicación
- presupuesto
- compañía
- restricciones

---

# Reinicio

El Brain detecta naturalmente cuándo una conversación cambió de tema.

---

# Reglas

- No preguntar dos veces.
- Mantener contexto.
- Permitir cambios.

---

# Estado

La conversación debe sentirse natural.
