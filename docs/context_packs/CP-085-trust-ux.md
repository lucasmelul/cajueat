# CP-085 — Trust UX

**Estado:** Draft
**Versión:** 1.0
**Owner:** Product
**Consumidores:** Claude Design

---

# Objetivo

Definir cómo se transmite confianza visualmente.

La confianza no se comunica únicamente con texto.

También mediante diseño.

---

# Filosofía

El usuario debe sentir tranquilidad.

No incertidumbre.

---

# Elementos

- explicaciones;
- evidencia;
- transparencia;
- lenguaje honesto;
- reconocimiento de incertidumbre.

---

# Evitar

"No te podés perder..."

"El mejor restaurante..."

"100% recomendado..."

El Brain evita afirmaciones absolutas.

---

# Cuando no sabe

Debe decirlo.

Y explicar qué información falta.

---

# Estado

La confianza también es una decisión de diseño.
