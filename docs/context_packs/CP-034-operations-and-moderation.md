# CP-034 — Operations & Moderation

**Estado:** Draft
**Versión:** 1.0
**Owner:** Product
**Consumidores:** Claude Code

---

# Objetivo

Definir cómo opera CajuEat internamente.

---

# Filosofía

La operación debe ser mínima.

El Brain debería resolver la mayor cantidad posible de tareas automáticamente.

Los operadores humanos intervienen únicamente cuando es necesario.

---

# Funciones

- revisar conflictos;
- aprobar cambios sensibles;
- administrar usuarios;
- moderar contenido;
- validar nuevas fuentes;
- supervisar calidad.

---

# Moderación

No buscamos moderar opiniones.

Buscamos proteger el conocimiento.

---

# Casos

- spam;
- información falsa;
- contenido duplicado;
- abuso;
- vandalismo.

---

# Automatización

Toda tarea repetitiva debe terminar siendo realizada por el Brain.

---

# Reglas

- Operación mínima.
- Automatización máxima.

---

# Documentos Derivados

- operations.md
- moderation.md

---

# Estado

La operación debe escalar mucho más lento que la comunidad.
