# Objetivo

Definir cómo las personas descubren lugares dentro de CajuEat.

La búsqueda es solamente uno de los mecanismos.

Nuestro objetivo es que descubrir sea incluso más importante que buscar.

---

# Filosofía

Las mejores experiencias gastronómicas muchas veces aparecen cuando el usuario no estaba buscando exactamente eso.

CajuEat debe favorecer el descubrimiento inteligente.

No únicamente responder consultas.
