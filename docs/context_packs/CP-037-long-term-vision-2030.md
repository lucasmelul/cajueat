# Objetivo

Documentar hacia dónde queremos llevar CajuEat durante los próximos años.

No es un roadmap.

Es una visión.

---

# Visión

Dentro de varios años queremos que CajuEat sea reconocido como el Brain gastronómico más confiable del mundo.

---

# Aspiraciones

- múltiples ciudades;
- múltiples países;
- API pública;
- SDK;
- asistentes de voz;
- agentes IA;
- integraciones;
- planificación inteligente;
- experiencias colaborativas.

---

# Lo que nunca debería cambiar

- Brain First.
- Trust First.
- Knowledge First.
- Conversation First.

---

# El verdadero activo

No será la aplicación.

No será la PWA.

No será el diseño.

Será el conocimiento acumulado por el Brain.

Ese será el mayor diferencial competitivo de CajuEat.

# Estado
