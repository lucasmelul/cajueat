# Objetivo

Definir cómo el Brain entiende el contexto.

Sin contexto no existen recomendaciones inteligentes.

---

# Filosofía

La misma pregunta puede tener respuestas completamente distintas.

Ejemplo.

"Quiero pizza."

No alcanza.

El Brain necesita entender.

- cuándo;
- dónde;
- con quién;
- para qué.

# Contexto Implícito

El Brain infiere.

No pregunta.
