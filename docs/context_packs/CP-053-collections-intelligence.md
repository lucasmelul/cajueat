# Objetivo

Definir cómo evolucionan las colecciones.

Las colecciones no son carpetas.

Son conocimiento.

---

# El Brain aprende

Guardar.

Mover.

Eliminar.

Ordenar.

Todo comunica información.

---

# Colecciones automáticas

El Brain puede generar.

- Favoritos ocultos.
- Cafeterías frecuentes.
- Lugares pendientes.
- Descubrimientos recientes.
- Restaurantes olvidados.

---

# Evolución

Las colecciones cambian automáticamente.

El Brain propone reorganizaciones.

---

# Estado

Las colecciones forman parte del sistema de memoria.
