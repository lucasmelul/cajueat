**Estado:** Draft
**Versión:** 1.0
**Owner:** Product
**Consumidores:** Claude Code

---

# Objetivo

Modelar zonas gastronómicas.

No solamente barrios administrativos.

---

# Filosofía

Una zona gastronómica tiene identidad.

No coincide necesariamente con los límites políticos.

---

# El Brain aprende

- polos gastronómicos;
- tendencias;
- horarios;
- tipos de cocina;
- evolución;
- eventos.

---

# Ejemplos

Palermo Hollywood.

Palermo Soho.

Villa Crespo.

Chacarita.

Microcentro.

Cada uno posee personalidad.

---

# Descubrimiento

El Brain puede recomendar zonas.

No solamente restaurantes.

---

# Objetivos

Responder preguntas como.

"¿Qué barrio me recomendás para salir hoy?"
