# CP-070 — Maps Strategy

**Estado:** Draft
**Versión:** 1.0
**Owner:** Product

---

# Objetivo

Documentar la estrategia de mapas.

---

# Decisión

Siempre que sea posible.

Utilizar OpenStreetMap y tecnologías abiertas.

Reducir dependencia de Google Maps.

---

# Filosofía

El mapa es infraestructura.

No diferencial.

El diferencial es el Brain.

---

# Google Maps

Utilizar únicamente cuando agregue valor.

Ejemplos.

- Directions.
- Places específicos.
- Información difícil de reemplazar.

---

# Datos valiosos

Si es posible integrar.

- horarios;
- popularidad;
- reviews;
- fotos;
- menú.

Siempre tratándolos como una fuente más.

Nunca como verdad absoluta.

---

# Estado

La estrategia prioriza independencia tecnológica y control de costos.
