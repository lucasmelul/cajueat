# CP-052 — Restaurant Comparison Engine

**Estado:** Draft
**Versión:** 1.0
**Owner:** Product

---

# Objetivo

Definir cómo el Brain compara alternativas.

Comparar es una de las funcionalidades más importantes del producto.

Muchas veces el usuario ya redujo las opciones.

Solo necesita ayuda para elegir.

---

# Filosofía

No queremos tablas.

Queremos criterio.

---

# Ejemplos

¿Cuál elegirías?

¿Vale la pena pagar más?

¿Para una primera cita cuál conviene?

¿Cuál tiene mejor café?

---

# Variables

- comida
- ambiente
- precio
- contexto
- distancia
- espera
- personalidad
- experiencia
- confianza

---

# Respuesta

Siempre debe existir una conclusión.

No solamente una comparación.

---

# Estado

Comparar es ayudar a decidir.
