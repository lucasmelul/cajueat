# CP-062 — Explainable AI UX

**Estado:** Draft
**Versión:** 1.0
**Owner:** Product
**Consumidores:** Claude Design

---

# Objetivo

Definir cómo mostrar el razonamiento del Brain.

---

# Filosofía

La explicación debe aumentar confianza.

Nunca complejidad.

---

# Capas

## Respuesta rápida

Una frase.

---

## ¿Por qué?

Tres razones.

---

## Profundizar

Toda la evidencia disponible.

---

# Ejemplo

Te recomiendo este restaurante porque:

- coincide con tus gustos;
- tiene excelente sushi tradicional;
- hoy presenta baja espera.

---

# Nunca mostrar

- scores internos;
- fórmulas;
- lenguaje técnico.

---

# Estado

La explicabilidad constituye una funcionalidad de UX, no solamente del Brain.
