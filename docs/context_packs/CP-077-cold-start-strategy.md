# CP-077 — Cold Start Strategy

**Estado:** Draft
**Versión:** 1.0
**Owner:** Product

---

# Objetivo

Definir cómo CajuEat genera valor cuando todavía sabe muy poco.

Todo producto basado en IA enfrenta el problema del Cold Start.

El usuario nuevo no tiene historial.

El Brain todavía no conoce sus gustos.

Y aun así debe sorprenderlo desde el primer minuto.

---

# Filosofía

El usuario nunca debería sentir que está entrenando a la IA.

Es el Brain quien debe hacer el esfuerzo.

No el usuario.

---

# Estrategia

El Brain combina:

- contexto actual;
- ubicación;
- horario;
- clima;
- conocimiento global;
- tendencias;
- restaurantes destacados;
- preguntas mínimas.

---

# Primer minuto

Idealmente.

El usuario ya recibe una excelente recomendación.

Sin completar perfiles.

---

# Primer aprendizaje

La primera interacción ya genera memoria.

Ejemplos.

- restaurante abierto;
- conversación;
- guardado;
- comparación.

---

# Objetivo

Reducir al máximo el tiempo hasta la primera experiencia exitosa.

---

# Estado

El Cold Start es uno de los principales desafíos del MVP.
