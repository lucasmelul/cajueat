# Objetivo

Definir cómo el Brain adapta toda la experiencia a cada usuario.

No queremos personalizar únicamente recomendaciones.

Queremos personalizar toda la experiencia.

Cada usuario debería sentir que CajuEat fue construido para él.

---

# Filosofía

No existen dos usuarios iguales.

Por lo tanto.

No deberían existir dos Home iguales.

Dos mapas iguales.

Dos rankings iguales.

Dos conversaciones iguales.

---

# Principios

## Personalización progresiva

El Brain aprende con el tiempo.

Nunca necesita un formulario de 30 preguntas.

---

## Transparente

El usuario puede entender por qué recibe determinadas recomendaciones.

---

## Adaptativa

La personalización cambia.

No queda congelada.

---

## Contextual

El mismo usuario puede recibir recomendaciones completamente distintas dependiendo del momento.
