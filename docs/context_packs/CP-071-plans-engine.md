# Objetivo

Definir cómo el Brain construye planes gastronómicos completos.

No queremos recomendar un restaurante.

Queremos recomendar una experiencia completa.

---

# Filosofía

Una salida no empieza cuando llegás al restaurante.

Empieza cuando decidís salir.

Y termina cuando volvés a tu casa.

El Brain debe acompañar todo ese recorrido.

# Plan Dinámico

Si cambia el contexto.

El plan cambia.

Ejemplo.

Empieza a llover.

↓

El Brain propone alternativas.
