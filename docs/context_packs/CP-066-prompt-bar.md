# CP-066 — Prompt Bar

**Estado:** Draft
**Versión:** 1.0
**Owner:** Product

---

# Objetivo

Definir la barra conversacional.

---

# Filosofía

No queremos un buscador.

Queremos una invitación a conversar.

---

# Ejemplos

¿Qué te gustaría comer?

Sorprendeme.

¿Dónde almorzamos?

Busco algo tranquilo.

Quiero un buen café.

---

# Capacidades

- texto;
- voz;
- imágenes;
- links.

---

# Reglas

Siempre visible.

Siempre accesible.

Nunca intimidante.

---

# Estado

La Prompt Bar representa la puerta de entrada al Brain.
