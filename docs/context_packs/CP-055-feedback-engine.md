# CP-055 — Feedback Engine

**Estado:** Draft
**Versión:** 1.0
**Owner:** Product

---

# Objetivo

Definir cómo el Brain obtiene feedback.

No queremos encuestas.

Queremos conversaciones.

---

# Cuándo preguntar

Después de una experiencia.

Nunca antes.

---

# Qué preguntar

Solo aquello que aporte conocimiento.

Ejemplos.

- ¿Volverías?
- ¿Qué fue lo mejor?
- ¿Esperaste mucho?
- ¿Qué pedirías nuevamente?

---

# Adaptación

El Brain cambia las preguntas según.

- restaurante
- contexto
- conocimiento existente

---

# Objetivo

Cada respuesta debe mejorar el Brain.

---

# Estado

El Feedback Engine constituye el principal mecanismo de aprendizaje continuo.
