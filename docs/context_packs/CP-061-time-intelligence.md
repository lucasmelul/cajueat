# Objetivo

Modelar el tiempo como una variable de primer nivel.

---

# Filosofía

Un restaurante cambia según:

- hora;
- día;
- estación;
- clima;
- eventos.

---

# El Brain aprende

- mejores horarios;
- tiempos de espera;
- cambios estacionales;
- días ideales;
- eventos especiales.

---

# Ejemplo

Excelente brunch.

↓

Solo domingos.

---

# Objetivo

Recomendar el momento ideal.

No solamente el lugar ideal.

---

# Estado

El tiempo modifica toda recomendación.
