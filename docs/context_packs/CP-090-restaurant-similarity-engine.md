# CP-090 — Restaurant Similarity Engine

**Estado:** Draft
**Versión:** 1.0
**Owner:** Product

---

# Objetivo

Definir cómo el Brain encuentra restaurantes similares.

---

# Filosofía

Dos restaurantes no son similares porque sirvan sushi.

Son similares porque generan experiencias parecidas.

---

# Variables

- propuesta;
- ambiente;
- chef;
- público;
- ticket;
- personalidad;
- experiencia;
- contexto.

---

# Ejemplo

Dos bares pueden parecer distintos.

Pero ambos ser ideales para una primera cita.

El Brain los considera similares.

---

# Estado

La similitud es semántica.

No categórica.
