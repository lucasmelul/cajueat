# Objetivo

Definir cómo funciona la búsqueda.

La búsqueda no devuelve resultados.

Devuelve respuestas.

---

# Filosofía

Buscar:

"Sushi"

No significa:

Mostrar todos los restaurantes japoneses.

Significa:

Entender qué intenta resolver el usuario.

---

## Conversación

Antes que filtros.

## Contexto

# Autocompletado

Debe sugerir.

No solamente completar texto.

---

# Sin resultados

Nunca mostrar.

"No encontramos resultados."

El Brain propone alternativas.

---

# Reglas

- Buscar intención.
- No palabras.
- Siempre ofrecer alternativas.
