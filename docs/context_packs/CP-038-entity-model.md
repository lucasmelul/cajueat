# Objetivo

Definir todas las entidades que existen dentro del universo CajuEat.

Este documento constituye la ontología de alto nivel del Brain.

No define tablas.

No define bases de datos.

Define conceptos.

# Filosofía

Todo lo que el Brain conoce debe estar representado mediante entidades y relaciones.

El Brain nunca piensa en registros.

Piensa en conceptos.
