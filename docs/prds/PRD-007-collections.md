# Objetivo

Permitir que los usuarios organicen su mundo gastronómico.

Pero mucho más importante.

Permitir que el Brain aprenda de esa organización.

---

# Problema

Guardar un restaurante hoy no significa nada.

Google Maps solamente guarda.

Nosotros queremos interpretar.

---

# Hipótesis

Cada restaurante guardado comunica preferencias.

El Brain debe aprender incluso cuando el usuario no escribe nada.

---

# Filosofía

Las colecciones no son carpetas.

Son conocimiento.
