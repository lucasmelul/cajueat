# PRD-012 — Trust Engine

**Status:** Draft
**Priority:** P0

---

# Objetivo

Administrar la confianza de todo el conocimiento.

---

# Filosofía

El Brain debe saber exactamente qué sabe.

Y qué todavía no sabe.

---

# Funciones

Calcular confianza.

Actualizar confianza.

Detectar contradicciones.

Detectar información vieja.

Solicitar evidencia.

---

# Nunca hacer

Asumir.

Inventar.

Ocultar incertidumbre.

---

# Acceptance Criteria

Toda afirmación importante posee:

- origen
- confianza
- evidencia
- fecha
