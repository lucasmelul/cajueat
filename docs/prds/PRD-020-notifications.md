# PRD-020 — Notifications

**Status:** Draft
**Priority:** P2

---

# Objetivo

Enviar únicamente notificaciones útiles.

---

# Casos

Nuevo descubrimiento.

Evento.

Recordatorio.

Plan.

Restaurante reabierto.

Cambio importante.

---

# Nunca enviar

Spam.

Promociones irrelevantes.

Notificaciones masivas.

---

# Acceptance Criteria

Cada notificación debe responder:

"¿Por qué vale la pena interrumpir al usuario ahora?"
